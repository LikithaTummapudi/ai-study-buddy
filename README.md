# 📚 AI Study Buddy

AI Study Buddy is an AI-powered web application designed to help students learn more efficiently. Instead of spending time searching through lengthy resources, students can instantly understand concepts, summarize notes, generate quizzes, and create flashcards using Google's Gemini AI.

## ✨ Features

* 📘 **Explain Topics** – Understand difficult concepts in simple student-friendly language.
* 📝 **Summarize Notes** – Convert long notes into concise summaries.
* ❓ **Generate Quizzes** – Create practice questions for quick revision.
* 🧠 **Create Flashcards** – Generate flashcards for effective memorization.

## 💻 Installation (Run Locally)

1. Clone the repository.

```bash
git clone https://github.com/LikithaTummapudi/ai-study-buddy.git
```

2. Install dependencies.

```bash
pip install -r requirements.txt
```

3. Create a `.env` file and add your Gemini API key.

```env
GEMINI_API_KEY=your_api_key
```

4. Run the application.

```bash
python app.py
```

5. Open your browser and visit:

```text
http://127.0.0.1:5000
```
